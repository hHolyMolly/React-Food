# Разработка интернет-магазина - REACT FOOD

REACT FOOD - это мной лично разработанный проект, начиная с дизайна и заканчивая фронт-енд разработкой.
Дизайн разрабатывался в Figma, а фронт-енд написан на фреймворке ReactJS, в качестве бек-енда использовал сервис [MockAPI](https://mockapi.io/).
Всё взаимодействие фронта и бека построено на API запросах и в качестве стейт менеджера использован Redux (RTK).

### Используемый стек технологий:

- ReactJS + hooks;
- Redux (RTK);
- SASS(SCSS)
- MockAPI
- Axios;
- ClassNames;
- Swiper;
- Lodash;
